1️- What is the difference between var, let, and const?
var: Function-scoped, can be redeclared and updated.
let: Block-scoped, can be updated but not redeclared.
const: Block-scoped, cannot be updated or redeclared.
2️- What is the spread operator (...)?
It expands an iterable (like an array) into individual elements. Used for copying or merging arrays/objects.
Example: const newArr = [...oldArr]
3️- What is the difference between map(), filter(), and forEach()?
map(): Transforms each element and returns a new array.
filter(): Selects elements based on a condition and returns a new array.
forEach(): Loops through elements but returns nothing.
4️- What is an arrow function?
A modern, shorter syntax for writing functions.
Example: const add = (a, b) => a + b
5️- What are template literals?
Strings wrapped in backticks (`) that allow embedded expressions using ${}.
Example: `Hello ${name}`
